ASE node
========

Agent Simulation Environment node

Requirements
------------
- java 1.7
- maven
- git

Download
--------
`git clone https://github.com/kolinger/ase-proxy.git`

Build
-----
`bin/build.bat` or `bin/build.sh`

Configuration
-------------
In `target/configuration.xml`

Running
-------
`bin/run.bat` or `bin/run.sh`
